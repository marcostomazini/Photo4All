## Sobre
Aplicativo Para Firefox OS, Para Tranmissão de Imagens em Tempo Real (Projetor de Imagens)

## Como Usar
Acessar o endereço http://photo4ffx.herokuapp.com
Abrir o Aplicativo no Celular e Ler o Qr Code do Site Aberto No Passo Acima ou Inserir a ID
Selecionar a Imagem e a Mesma Será Retransmitida ao Site Que Foi Aberto. 
Funciona em SmartTV, PS3 em todo aparelho que rodar javascript e estiver conectado a nuvem.
